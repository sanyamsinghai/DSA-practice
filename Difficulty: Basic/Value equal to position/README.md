<p><span style="font-size: 18px;"><strong>Note:</strong> There can be more than one element in the array which have the same value as its position. You need to include the position of every such element.</span></p>

<strong>Explanation:</strong> </span><span style="font-size: 18px;">Here, arr[2] = 2 exists here </span><span style="font-size: 18px;">and arr[4] = 4 exists.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [1]
<strong>Output:</strong> [1]
<strong>Explanation:</strong> Here arr[1] = 1 exists.</span></pre>
